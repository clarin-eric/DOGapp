Welcome to CLARIN ERIC Digital Object Gateway. The goal of DOG is to ease access to referenced resources in metadata. Currently, DOG supports 3 primary functionalities:
  • sniff - checks if input PID points to a registered repository
  • fetch - fetches all PIDs of referenced resources in the metadata
  • is pid - checks if input string is a DOG parsable PID
