Welcome to CLARIN-ERIC Digital Object Gate. The goal of DOG is to ease access to referenced resources in metadata. Currently, DOG supports 4 functionalities:
  • sniff - checks if input PID points to a registered repository
  • fetch - fetches all PIDs of references resources in the metadata
  • is pid - checks if input string is a DOG parsable PID
  • expand data type - returns MIME Data Type taxonomy from Data Type Registry
